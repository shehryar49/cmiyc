# cmiyc
A small easy to use tool to add noise to ASCII text. It takes as input a ascii text file and outputs an UTF-8 file by adding some noise.
It's purpose? To evade plagiarism detection tools. Input and Output text look same in appearance!

## INSTALLATION & SETUP
Just compile the cmiyc.cpp

## USAGE
```
./cmiyc input.txt
```

A new file **generated.txt** will be created.
